# Api para poder comunicarse con mikrotik

## Inicializacion:
  instalar el entorno virtual con
  ```javascript
  py -m pip install virtualenv
  ```
  inicializar con:
  ```javascript
  py -m virtualenv pythonmodule
  ```
  Ejecutar con:
  ```javascript
  pythonmodule/Scripts/Activate.ps1
  ```

## Dependencias
  Comando pip install
  1. laiarturs-ros_api
  2. flask

## Ejecutar servidor
  ```javascript
  python app/main.py
  ```
